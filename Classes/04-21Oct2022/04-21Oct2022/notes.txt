HTML Attributes:

Attribute: is a keyword, that will provide some extra capabilities to an element.

it can contains one or more values.

<input id='username' style='color:green; background-color:red; height:10px'>

2 Types of attributes are available in HTML

1. Global attributes
2. Element specific attributes

Global attributes:

id, class, style

<body id='username'> <p id='username'> <h1 id='username'>


Element specific attributes:

type, placeholder

<input id="uname" type="text" placeholder="please enter username">

HTML Block level elements, Inline elements:

Block level elements: will be srarted in new line and occupies complete width of web page.
Ex: <div> <p> <h1> etc....

Inline elements: will be placed on the same line and occupies width based content provided.
Ex: <a> <span> <img>
