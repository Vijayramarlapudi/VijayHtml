Operators:

An operator is a special symbol/keyword that performs some action.

ex: +,- typeof, instanceof, <, >, ?: .............



Arithmetic Operators:

+, -, *, /, %, --, ++

-- : decrement operator --> decrements the value by 1
++ : increment operator --> increments the value by 1



Compparision Operators:

The comparision operators will always result a boolean value.

>, <, >=, <=, !=(Not equels to), ==(Value comparison), ===(Type and value comparision), ?:(Conditional Operator)


Logical Operators:

AND(&&), OR(||), NOT(!)
